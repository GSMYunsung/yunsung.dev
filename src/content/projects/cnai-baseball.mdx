## 개요

8만 명 이상이 이용하는 야구 팬 플랫폼의 전체 구조를 파악하고 유지보수를 담당했습니다. 레거시 코드에서 비롯된 데이터 불일치 문제와 이벤트 추첨 로직의 설계 결함을 발견하고 수정했으며, 수동으로 운영되던 멤버십 발급 체계를 자동화했습니다.

## 주요 작업

- 레거시 코드로 인해 **티켓 사이트 정보와 플랫폼 데이터가 불일치하던 문제** 원인 파악 및 수정
- 회원 티켓 구매 시 포인트가 정상 지급되지 않던 로직 오류 수정
- 화면 UI 퍼블리싱 및 비즈니스 로직 연결
- **아이핀·PASS 인증 오류** 해결

## 기술적 기여

**이벤트 추첨 시스템 재설계**

기존 로직은 이전에 당첨된 이력이 있는 사용자를 추첨 대상에서 제외하는 방식이었습니다. 이 때문에 **당첨 확률이 구매 이력에 따라 달라지는 공정성 문제**가 있었고, 반복 구매 사용자일수록 오히려 불이익을 받는 역설적인 구조였습니다. **모든 티켓 구매자를 동등한 조건으로 포함하는 랜덤 추첨 시스템**으로 재설계해 공정성을 확보했습니다.

**추첨 명단 엑셀 업로드 자동화**

기존에는 추첨 대상 명단을 관리자가 백엔드 시스템에 직접 수작업으로 입력해야 했습니다. **엑셀 파일을 업로드하면 데이터를 파싱해 백엔드 서버에 자동으로 전송**하는 기능을 구현해, 명단 등록 과정에서 발생하던 수동 작업과 입력 오류를 줄였습니다. 기존에 **하루 이틀씩 소요되던 이벤트 명단 관리 작업이 한 시간 안에 완료**될 수 있게 되었고, 삼성 라이온즈 관계자로부터 직접 긍정적인 피드백을 받았습니다.

**멤버십 회원가입 자동화**

기존에는 관리자가 백엔드 시스템에 멤버십 코드를 직접 입력한 뒤 사용자에게 전달하는 수동 방식으로 운영됐습니다. **백엔드에서 코드를 자동 생성·발급하는 로직으로 전환**해, 관리자 개입 없이 가입 시 코드가 자동 발송되도록 개선했습니다.
